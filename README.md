# Crop-Production-Analysis-in-India
## 📌 Project Overview
This project is an interactive **Power BI dashboard** designed to analyze agricultural data across India. It provides insights into **crop production, cultivated area, yield, and regional performance** using dynamic and interactive visualizations.

The dashboard is structured into **four key analytical pages**, enabling users to explore data from high-level summaries to detailed trends and comparisons.

---

## 🎯 Objectives
- Analyze agricultural trends across years and regions  
- Identify top-performing states and crops  
- Understand the relationship between area, production, and yield  
- Provide interactive dashboards for better decision-making  

---

## 🛠️ Tools & Technologies
- **Power BI Desktop** – Dashboard creation  
- **DAX (Data Analysis Expressions)** – Calculated measures  
- **Power Query** – Data cleaning and transformation  
- **Dataset** – Agriculture dataset  

---

## 📊 Dashboard Pages Overview

### 📍 1. Overview Dashboard
This page provides a **high-level summary** of agricultural performance.

#### 🔑 KPIs:
- **Total Crop Area:** 6.51M  
- **Total Production:** 10.27M  
- **Total Yield:** 1.58K  

#### 📊 Visuals:
- State-wise Area & Production comparison  
- Year-wise trend analysis  
- Crop distribution (Donut chart)  

👉 **Insight:** Agriculture is unevenly distributed, with some states contributing significantly more than others.

---

### 🌍 2. State & Regional Analysis
This page focuses on **geographical performance across states**.

#### 📊 Visuals:
- Top states by crop area  
- Top states by production  
- India map visualization  
- Year-wise trends by state  
- Detailed data table  

👉 **Insight:** Some states have large agricultural areas but lower production, indicating differences in efficiency.

---

### 🌱 3. Crop-wise Analysis
This page analyzes **crop-level performance**.

#### 📊 Visuals:
- Crop-wise area distribution  
- Crop-wise production comparison  
- Pie charts showing crop contribution  
- Year-wise trends (Area vs Production)  

👉 **Insight:** A few crops dominate both land usage and production.

---

### 📈 4. Trend & Performance Analysis
This page focuses on **long-term trends and growth patterns**.

#### 📊 Visuals:
- Year-wise crop area trend  
- Production trend over time  
- Top crops by area  
- Combined trend analysis  

👉 **Insight:** Production shows a steady increase, but growth is influenced by both area and efficiency.
